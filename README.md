# zxhud — FiveM HUD Resource

A custom FiveM HUD resource featuring a dark cyberpunk UI with teal (#385f5b) accent color, minimap integration, camera status indicator, and smooth animations.

## Features
- Dark cyberpunk theme with teal accent
- Custom minimap overlay
- Camera status indicator
- Gear, speed, fuel, and engine displays
- Smooth CSS animations and glassmorphism panels

## Installation
1. Drop the `zxhud` folder into your `resources` directory
2. Add `ensure zxhud` to your `server.cfg`
3. Restart your server

## Dependencies
- `/assetpacks`
